## 📋 CÁCH SỬ DỤNG CÁC PROMPTS NÀY

### Midjourney:
```
1. Vào Discord/Midjourney web
2. /imagine [paste prompt]
3. Chọn version: --v 6 (đã include trong prompt)
4. Generate 4 variants → upscale cái đẹp nhất
5. Thêm: --style raw (nếu muốn ít artistic hơn, realistic hơn)
```

### DALL-E 3 (ChatGPT):
```
1. Bỏ phần "--ar 3:4 --v 6" ở cuối
2. Thêm vào đầu: "Create a photorealistic product photo of:"
3. Kết quả thường mượt hơn Midjourney về texture thực
```

### Adobe Firefly:
```
1. Firefly rất tốt cho textile/fabric textures
2. Dùng "Reference image" = ảnh template trắng từ user
3. Paste prompt vào "Text to Image"
```

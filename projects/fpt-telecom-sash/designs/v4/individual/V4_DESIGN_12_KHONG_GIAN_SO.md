## CONCEPT

An abstract network diagram representing FPT's digital ecosystem—interconnected nodes (circles) linked by metallic thread lines. Some nodes contain small tech icons (router symbol, cloud, server icon, etc.). The design represents how FPT connects Vietnam to the digital world.

Color gradient suggests evolution from foundational darkness (1997) through current cyan connectivity to future neon-bright possibilities.

## CRAFTSMANSHIP DETAILS

### Embroidery:

- **Network Nodes**: 8-12 circles, various sizes (0.5cm to 2cm diameter)
  - Positioned throughout panel in seemingly organic pattern (actually mathematically balanced)
  - Each filled with different shades (some navy, some cyan, some green)
  - Executed in satin stitch with subtle shading

- **Connection Lines**: Silver and gold metallic thread using couching technique
  - Links nodes in continuous network pattern
  - Some lines heavy and obvious, some whisper-thin
  - Creates sense of flowing data and information

- **Tech Icons** (optional, in select nodes):
  - Small router symbol
  - Cloud icon
  - Server/tower symbol
  - Lock icon (security)
  - All approximately 0.3-0.5cm, in fine detail

- **Layering**: Nodes appear to float at different depth planes due to shading

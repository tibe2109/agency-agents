## TECHNICAL SPECIFICATIONS FOR PRODUCTION

### Sash Dimensions
```
Total Length        : ~150 cm (standard clergy stole length)
Width per Panel     : ~18 cm (two panels for shoulder wear)
Aspect Ratio        : 2:3 (standard sash proportion)
Wear Format         : Draped across shoulders, hanging front panels
```

### Color Accuracy Requirements
```
Standard Tolerance  : ±5% color variance from hex codes
Critical Elements   : Gradients must transition smoothly (±10 steps max)
Metallic Thread     : Must provide proper sparkle/reflection
Dyeing Method       : Hand-painted OR chemical dyeing (per design)
```

### Embroidery Specifications (A, B, C Series)
```
Stitch Types        : Satin, split, long-and-short, backstitch, couching
Metallic Integration: Max 30% of embroidery area in metallic
Thread Gauge        : 2-4 ply silk embroidery thread
Density             : 20-25 stitches per cm
Relief Height       : 2-4mm (3D effect, where applicable)
```

### Embroidery Specifications (D Series - Gradient Only)
```
Technique           : Hand-painted acid dye gradient on silk
Gradient Smoothness : ZERO banding artifacts
Transition Zones    : 5-7 distinct color zones, seamlessly blended
Testing Required    : Color proof sample before full production
```

### Embroidery Specifications (E Series - Master Craft)
```
Technique Variety   : 2-5 different techniques per design
Relief Height       : 4-8mm (dimensional, tactile)
Thread Types        : Mix of silk, metallic, specialty threads
Production Time     : 40-80 hours per sash
Quality Level       : Museum-grade, individual artisan work
```

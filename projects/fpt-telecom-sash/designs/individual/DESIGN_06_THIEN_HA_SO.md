## 📐 BỐ CỤC & HỌA TIẾT

```
NODES        : 50-70 nodes tổng
               5 nodes lớn (Ø4-5mm) = 5 cột mốc: FOX, AAG, Quang Hóa, Vươn Mình, FTEL30
               Các nodes khác: 2-3mm, flat-stitched
               Density gradient: dày ở trên (gần V-notch), thưa ở dưới

ĐƯỜNG NỐI   : Lines kết nối giữa nodes
               Primary: 1.2mm · Secondary: 0.6mm · Tertiary: 0.3mm
               Tổng thể: Organic, không phải grid

NODE TÂM     : Node lớn nhất (6mm) ở giữa-trên khăn
               Hào quang nhẹ xung quanh → focal point

EASTER EGG   : "50" ẩn trong pattern (nodes tạo thành số 50 khi nhìn xa)

UV LAYER     : UV-reactive ink in phía dưới — invisible bình thường
               Dưới UV: Toàn bộ network sáng như thiên hà James Webb

LOGO         : Đặt tại node chính dưới-giữa → Logo trở thành một node trong mạng
```

## 🔧 KỸ THUẬT SẢN XUẤT

```
UV PRINT      : Digital print với UV-reactive ink (nền)
                In trước, thêu sau
                UV ink on silk: cần test kỹ — silk có thể nhạy với UV process

NODE THỦ CÔNG: Paillettes / sequins vàng tay (50+ nodes lớn)
               Mỗi node = 1 sequin đính tay

KỸ THUẬT 2 LỚP:
  Layer 1: UV pattern print (invisible thường, sáng dưới UV)
  Layer 2: Gold sequin nodes (visible thường + UV)

VẢI           : Atlas Satin Silk — controlled low-reflection, deep space feel

RỦI RO SX    : UV ink process phức tạp — cần test trước
               Nếu UV effect không đạt: fallback về design without UV
```
